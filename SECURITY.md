# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Ckourse, please report it privately via GitHub's [Report a vulnerability](https://github.com/redaantar/ckourse/security/advisories/new) flow. Do not open a public issue for security problems.

Please include:

- A description of the issue and its impact
- Steps to reproduce
- Affected Ckourse version and OS

You can expect an initial response within a few days. Once the issue is confirmed and a fix is available, a coordinated disclosure will be published as a GitHub security advisory.

## Supported Versions

Only the latest release receives security fixes.
